# Toolbox Catalog

Use this catalog to identify which toolbox contains the skills needed for a user request.

| Toolbox | Keywords | Description |
| :--- | :--- | :--- |
| **ProjectDiscovery** | layers, extent, project, map, list, what is in, current layer, active layer | Basic project awareness and map metadata. |
| **DataInspection** | fields, attributes, columns, crs, projection, coordinate system, data, rows, records, values | Detailed metadata about layers and their structures. |
| **SelectionTools** | select, filter, query, where, expression, location, intersect, contains, clear, deselect | Spatial and attribute-based selection. |
| **LayerStyling** | color, colour, style, symbology, renderer, appearance, zoom, transparency, transparent, opacity | Changing how layers look on the map. |
| **VectorProcessing** | buffer, clip, reproject, join, merge, dissolve, extract, difference, split, fix, convex hull, polygons, lines, centroid, intersection, bounding box | Core vector geoprocessing algorithms. |
| **RasterAnalysis** | slope, hillshade, aspect, raster, dem, elevation, calculator, mask | Tools for processing and analyzing raster data. |
| **WebServices** | wms, wfs, xyz, tiles, google maps, osm, remote, server | Adding layers from web-based services. |
| **TerrainAnalysis** | tin, idw, interpolation, contours, relief, rugosity | Advanced terrain and surface modeling. |
| **ExportTools** | export, save as, pdf, csv, geojson, shapefile, database, png, image | Exporting layers and map layouts to files. |
